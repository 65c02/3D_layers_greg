Ce tool permet de decomposer une image en couche 3D pour permettre une impression 3D multicolor

cliquez sur le bouton "Open Image" pour ouvrir une image
cliquez sur le bouton "Generate" pour generer les couches
cliquez sur le bouton "Export" pour exporter les couches

vous pouvez changer l'ordre des couches en faisant un drag n drop de la couche dans la liste

vous pouvez choisir l'épaisseur des couches
vous pouvez choisir la taille de l'image à impression
